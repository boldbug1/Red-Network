Installation Manual

Run the following commands directly inside your terminal or deployment environment environment setup scripts.

Step 1: Fetch core binaries
$ git clone https://github.com/your-repo/red-engine.git
Step 2: Enter directory & Build
$ cd red-engine && npm install
Step 3: Run Engine Environment
$ npm run dev
